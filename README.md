# Gastos App

Proyecto simple para aprender:

- React
- .NET Web API
- GitHub
- Docker
- Kubernetes

## Estructura del proyecto

- `frontend`: aplicación React
- `backend`: API en ASP.NET Core

## Requisitos

- Node.js
- npm
- .NET SDK
- Git

## Cómo levantar el backend

```bash
cd backend
dotnet run --urls=http://localhost:5000
